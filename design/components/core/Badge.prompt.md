**Badge** — a tiny count or status marker. `gold` flags an unsubmitted note, `accent` shows an annotation count, `plain` is bare coloured text.

```jsx
<Badge tone="gold">未交</Badge>
<Badge tone="accent">3</Badge>
<Badge tone="plain">✎3</Badge>
```

Tones: `accent` (default periwinkle pill), `gold` (unsubmitted), `rose` (human), `plain` (no pill).
