**Highlight** — inline passage mark rendered as an underline wash: gold for the human, ice-blue for 小奏. Wrap the highlighted run inside the reading `Sheet` text.

```jsx
<Highlight by="human">this line matters because I found myself in it</Highlight>
<Highlight by="ai" active>a passage 小奏 also stopped at</Highlight>
```

`by`: `human` (gold) / `ai` (ice). `active` deepens the wash for the selected mark.
